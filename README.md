
# Epics - Career Portal for Class 9-10 Students

## Overview
Epics is a Flutter-based career exploration platform designed for class 9-10 students. It helps young learners discover career paths, understand industry requirements, and gain insights into various professions.

## Features
- **Career Exploration**: Browse different career options and industries
- **Career Guidance**: Learn skills and qualifications needed for various paths
- **Interactive Learning**: Engaging content to help students make informed decisions
- **User-Friendly Interface**: Designed specifically for teenage users

## Tech Stack
- **Framework**: Flutter
- **Language**: Dart
- **Platform**: Cross-platform (iOS, Android, Web)

## Getting Started
1. Ensure Flutter is installed
2. Clone the repository
3. Run `flutter pub get`
4. Run `flutter run`
